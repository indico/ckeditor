
by

```JavaScript
const subprocess = childProcess.spawnSync( 'npm', yarnArguments, {
```

```JavaScript
const packages = childProcess.execSync( 'ls -1 packages', {
```

by

```JavaScript
const packages = childProcess.execSync( 'dir /b packages', {
```

and

```JavaScript
} ).toString().trim().split( '\n' );
```

by
